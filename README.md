# 🎬 Movie Recommendation System

An AI-powered Movie Recommendation System that provides personalized movie suggestions using Content-Based Filtering, Collaborative Filtering (SVD Matrix Factorization), and Hybrid Recommendation techniques. The project leverages machine learning algorithms to recommend movies based on user preferences and historical rating data.

---

## 📌 Overview

Recommendation systems are widely used by streaming platforms such as Netflix, Amazon Prime, and Disney+ to improve user experience. This project demonstrates how machine learning can be used to build a personalized movie recommendation engine.

The system combines multiple recommendation approaches to generate accurate and relevant movie suggestions for users.

## 🧠 Machine Learning Techniques

### 1. Content-Based Filtering
Recommends movies based on similarity between movie genres and attributes using cosine similarity.

### 2. Collaborative Filtering
Uses user rating patterns to predict preferences through SVD (Singular Value Decomposition).

### 3. Hybrid Recommendation
Combines Content-Based and Collaborative Filtering approaches to improve recommendation accuracy.
